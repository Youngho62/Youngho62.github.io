졸업하고 만든 두 번째 프로젝트입니다. 스프링부트와 JPA, MySql, Thymeleaf, 'Amado - Free Multipage e-commerce Template' 무료 부트스트랩 템플릿을 이용하여 만든 쇼핑몰입니다.   AWS의 EC2를 이용하여 웹서버 호스팅을 하였고, 도메인을 구매해 DNS 서비스를 이용하였습니다.
해당 프로젝트는 첫 번째 프로젝트에서 대부분의 기능을 이용하였고 다양한 기능들을 더 추가하였습니다.  우선 홈페이지(index.html)에는 상품들을 조회 수 순으로 정렬하여 11개가 보이게 하였습니다.
그리고 회원가입을 할 때는 다음(kakako) 주소 API를 이용하여 실제 주소를 검색할 수 있게 구현하였습니다.
상품 리스트(/product/list.html)페이지에서 제품들을 카테고리별로 조회할 수 있으며, 페이지네이션을 통해 기본으로 10개씩 보이게 하였고 사용자가 원하면 10, 20, 30, 40, 50개씩 볼 수 있는 select 태그를 삽입하였습니다.  또한 상품 사진을 보고 싶지 않으면 사진은 사라지게 할 수 있게 타일을 만들었습니다.
상품을 클릭하면 상품 상세조회(/product/view.html) 페이지가 나오며 원하는 개수대로 장바구니에 넣을 수 있습니다. 해당 페이지에서 제품 사진들을 클릭하면 원본 사진이 나타나고 아래에 있는 사진들을 클릭하면 main 사진과 위치가 바뀝니다.
상품 등록을 할 때 main 사진을 등록할 수 있게 하였고, Main 사진을 등록하지 않았을 경우 기본 사진으로 대체됩니다.
장바구니(cart/list.html)페이지내에서 상품 개수와 총가격들을 실시간으로 변경할 수 있게 AJAX를 이용하여 구현하였습니다. 결제할 때 포인트제도를 운영하고 있으며 포인트는 결제금액(배송비 제외)의 10%로 정하였습니다. 결제 방법으로는 카드결제, 계좌이체가 있고 카드결제는 'iamport'사의 결제 대행 API를 이용하여 실제 결제까지 구현하게 했습니다.  테스트용이므로 결제는 해당일 자정 전에 환불 됩니다.  또한 QnA 게시판을 구현해 사용자와 관리자 간에 자유로운 의사소통이 가능하게 구현하였습니다. 게시판은 기본적으로 CRUD 기능을 전부 구현 하였으며 댓글 기능 대신, 답글 작성이 가능하게 구현하였습니다. 답글은 계층형으로 작성되게 구현하였습니다. 마이페이지에는 회원정보수정, 주문내역, 나의 문의내역을 확인할 수 있게 구현했습니다.

게시판 구현

WebSite Go! - http://yhsboard.xyz
 GitHub Go! - https://github.com/Youngho62/bootjpa


졸업하고 만든 첫 프로젝트입니다. 스프링부트와 JPA, MySql, Thymeleaf 템플릿을 이용하여 만든 게시판입니다.
 AWS의 EC2를 이용하여 웹서버 호스팅을 하였고, 도메인을 구매해 DNS 서비스를 이용하였습니다.
해당 프로젝트는 스프링 시큐리티를 이용하여 로그인하며 데이터베이스로 회원을 관리합니다. 스프링 시큐리티를 배운 적은 있지만 직접 활용을 해본 적은 처음이라 매우 많은 어려움이 있었습니다. PasswordEncoder를 만들어 비밀번호를 암호화하고, CSRF Token을 이용하여 보안을 강화하는 것이 어려웠지만, 책과 인터넷을 통해 계속 찾아보고 프로젝트에 이용할 수 있었습니다.  회원정보를 수정할 수 있고, 프로필 사진을 등록할 수 있게 만들었습니다.
게시판은 기본적으로 CRUD 기능을 전부 갖고 있고, 댓글은 RestController를 이용하여 비동기 통신 방식인 AJAX로 구현 하였습니다. 댓글의 CRUD 기능은 모달창에서 이용 가능할 수 있게 했습니다.
게시판의 검색 기능은 다양한 검색 조건을 가지기 때문에 Querydsl을 이용하여 구현해보았습니다. Querydsl을 이용한 동적 SQL 처리 방법이 생각보다 적용이 잘되지 않고 이해가 되지 않았습니다. 이것 때문에 며칠 동안 진행이 되지 않아 정적인 쿼리문을 이용할까 고민했지만, 해당 Querydsl을 이용한 관련 책을 알게 되어 공부하고 따라 해본 결과 문제를 해결하게 되었습니다.
게시판의 글을 작성할 때 첨부 파일을 올릴 수 있고 여러 개의 파일을 동시에 등록할 수 있도록 구현하였습니다. 이미지 파일의 경우 기본적으로 썸네일이 보여지게 했고 클릭 시 원본 파일이 보일 수 있게 하였습니다. 일반 파일은 클릭 시 다운로드 될 수 있게 구현하였습니다.
